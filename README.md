## Installation

After cloning this repo, type in the terminal:

```bash
 yarn install
```

then cd into the client and:

```bash
 yarn install
```

then go to the server folder and:

then cd into the client and:

```bash
 yarn install
```

run the front-end and back-end with:

```bash
 npm start
```
(run in root folder)
